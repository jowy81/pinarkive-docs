# POST /files

Upload a single file. The file is added to IPFS and pinned in the cluster. Requires authentication (Bearer token or API key) and scope `files:write`.

## Endpoint

```
POST https://api.pinarkive.com/v3/files
```

## Description

- **Content-Type:** `multipart/form-data`
- **Body:** form field `file` (required). Optional: `cl` (cluster id), `timelock` (ISO 8601, premium only).
- **Response:** CID, clusterId, and optionally expiresAt.

## cURL

```bash
curl -X POST https://api.pinarkive.com/v3/files \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F file=@image.png
```

With optional cluster and timelock (premium):

```bash
curl -X POST https://api.pinarkive.com/v3/files \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F file=@document.pdf \
  -F cl=cl0-global \
  -F "timelock=2026-12-31T23:59:59Z"
```

## Python

```python
import requests

url = "https://api.pinarkive.com/v3/files"
headers = {"Authorization": f"Bearer {os.environ['PINARKIVE_API_KEY']}"}
files = {"file": open("image.png", "rb")}
data = {"cl": "cl0-global"}  # optional

r = requests.post(url, headers=headers, files=files, data=data)
r.raise_for_status()
print(r.json()["cid"], r.json()["clusterId"])
```

## Go

```go
package main

import (
	"bytes"
	"io"
	"mime/multipart"
	"net/http"
	"os"
)

func main() {
	url := "https://api.pinarkive.com/v3/files"
	apiKey := os.Getenv("PINARKIVE_API_KEY")

	var buf bytes.Buffer
	w := multipart.NewWriter(&buf)
	file, _ := os.Open("image.png")
	defer file.Close()
	part, _ := w.CreateFormFile("file", "image.png")
	io.Copy(part, file)
	w.WriteField("cl", "cl0-global") // optional
	w.Close()

	req, _ := http.NewRequest("POST", url, &buf)
	req.Header.Set("Authorization", "Bearer "+apiKey)
	req.Header.Set("Content-Type", w.FormDataContentType())

	resp, _ := http.DefaultClient.Do(req)
	defer resp.Body.Close()
	// parse JSON from resp.Body for cid, clusterId
}
```

## PHP

```php
<?php
$url = 'https://api.pinarkive.com/v3/files';
$apiKey = getenv('PINARKIVE_API_KEY');

$cfile = new CURLFile('image.png', 'image/png', 'image.png');
$post = ['file' => $cfile, 'cl' => 'cl0-global']; // cl optional

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $post);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer ' . $apiKey,
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);
$data = json_decode($response, true);
echo $data['cid'] . ' ' . $data['clusterId'];
```

## Example response

**Success (200/201)**

```json
{
  "cid": "bafybeigdyrzt5sfp7udm7hu76uh7y26nf3efuylqabf3oclgtqy55fbzdi",
  "clusterId": "cl0-global",
  "expiresAt": "2026-12-31T23:59:59.000Z"
}
```

| Field       | Type   | Description                          |
|------------|--------|--------------------------------------|
| `cid`      | string | IPFS content identifier             |
| `clusterId`| string | Cluster where the file was pinned   |
| `expiresAt`| string \| null | Expiration (timelock), if set |

**Error (4xx/5xx)**

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key"
}
```

Common codes: **401** (missing/invalid token), **403** (e.g. plan limit, timelock on free plan), **413** (file too large), **429** (rate limit).
