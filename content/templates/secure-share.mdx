# Secure Share Template

This template helps you build a **secure share** flow: users upload a file, optionally set an expiration, and get a shareable link. You can extend it with encryption or access control as needed.

## What's included

- Upload UI with optional expiration (timelock) field
- Backend call to Pinarkive `POST /upload` (or `POST /pin`) with `expires_at`
- Generation of a shareable gateway URL
- Optional: client-side encryption before upload, or signed URLs (implementation depends on template version)

## Getting started

1. Get the template from the [Pinarkive GitHub](https://github.com/pinarkive).
2. Install dependencies and set `PINARKIVE_API_KEY` in `.env`.
3. Run the app and use the upload form; set an expiration time to test timelocks.

## Key implementation details

- Use the [Upload](/api/upload) or [Pin](/api/pin) API with `expires_at` (ISO 8601) for time-limited links.
- Share the gateway URL: `https://gateway.pinarkive.com/ipfs/<cid>`.
- For “encrypted” sharing, the template may combine client-side encryption with upload; decrypt after download in the client.
