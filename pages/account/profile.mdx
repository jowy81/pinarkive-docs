# Profile

Account profile and settings are managed in the **Pinarkive dashboard**, not via the public Files API.

- **View or edit profile:** Log in at [pinarkive.com](https://pinarkive.com) and open your account or profile settings.
- **Change password, email, or 2FA:** Use the same dashboard settings.

The API does not expose a dedicated “profile” endpoint for public docs. For programmatic access to user info, check the dashboard or contact support for available account APIs.
