# Plans

Plans, quotas, and billing are managed in the **Pinarkive dashboard**.

- **View current plan:** Log in at [pinarkive.com](https://pinarkive.com) and open billing or plan settings.
- **Limits:** Storage, request limits, and features (e.g. timelocks, clusters) depend on your plan. The API may return **403** when a limit is exceeded (e.g. storage or upload size).
- **Upgrade or change plan:** Use the dashboard or contact support.

The public API does not expose a dedicated “plans” endpoint. For exact limits and pricing, see the dashboard or [pinarkive.com](https://pinarkive.com).
