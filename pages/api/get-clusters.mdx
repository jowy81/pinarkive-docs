# GET /clusters

List clusters allowed for your plan and their gateways (for viewing files). Requires authentication and scope `cluster:read`. The API endpoint is `GET /users/me/clusters`.

## Endpoint

```
GET https://api.pinarkive.com/v3/users/me/clusters
```

## Description

- No query parameters required.
- **Response:** Array of clusters; each includes `gateways` with `publicUrl` or `url` and `label`. Use `publicUrl` when present, otherwise `url`.

## cURL

```bash
curl -X GET "https://api.pinarkive.com/v3/users/me/clusters" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## JavaScript

```javascript
const response = await fetch('https://api.pinarkive.com/v3/users/me/clusters', {
  headers: {
    'Authorization': `Bearer ${process.env.PINARKIVE_API_KEY}`,
  },
});

const data = await response.json();
const gatewayUrl = data[0]?.gateways?.[0]?.publicUrl || data[0]?.gateways?.[0]?.url;
console.log('View file at:', `${gatewayUrl}ipfs/<cid>`);
```

## Python

```python
import os
import requests

url = "https://api.pinarkive.com/v3/users/me/clusters"
headers = {"Authorization": f"Bearer {os.environ['PINARKIVE_API_KEY']}"}

r = requests.get(url, headers=headers)
r.raise_for_status()
clusters = r.json()
for c in clusters:
    for gw in c.get("gateways", []):
        base = gw.get("publicUrl") or gw.get("url")
        print(c.get("clusterId"), base)
```

## PHP

```php
<?php
$url = 'https://api.pinarkive.com/v3/users/me/clusters';
$apiKey = getenv('PINARKIVE_API_KEY');

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer ' . $apiKey,
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);
$clusters = json_decode($response, true);
foreach ($clusters as $c) {
    foreach ($c['gateways'] ?? [] as $gw) {
        $base = $gw['publicUrl'] ?? $gw['url'];
        echo $c['clusterId'] . ' ' . $base . "\n";
    }
}
```

## Example response

**Success (200)**

```json
[
  {
    "clusterId": "cl0-global",
    "gateways": [
      {
        "publicUrl": "https://gateway.pinarkive.com/ipfs/",
        "url": "https://gateway.pinarkive.com/ipfs/",
        "label": "Primary",
        "order": 0
      }
    ]
  }
]
```

Gateway URLs are the **base** (ending with `/ipfs/`). Append the CID: `base + cid` → e.g. `https://gateway.pinarkive.com/ipfs/<cid>`.

**Error (4xx/5xx)**

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing API key"
}
```

Common codes: **401**, **403** (missing scope: `code: missing_scope`, `required`), **429** (rate limit; `retryAfter`, header `Retry-After`).
