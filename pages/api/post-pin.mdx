# POST /pin

Pin an existing CID (content already on IPFS). Requires authentication and scope `files:write`.

## Endpoint

```
POST https://api.pinarkive.com/v3/files/pin/:cid
```

Replace `:cid` with the IPFS content identifier (e.g. `bafybeigdyrzt5sfp7udm7hu76uh7y26nf3efuylqabf3oclgtqy55fbzdi`).

## Description

- **Content-Type:** `application/json`
- **Body (optional):** `originalName`, `customName`, `cl` (cluster id), `timelock` (ISO 8601, premium only).
- **Response:** CID, clusterId, and optionally expiresAt.

## cURL

```bash
curl -X POST "https://api.pinarkive.com/v3/files/pin/bafybeigdyrzt5sfp7udm7hu76uh7y26nf3efuylqabf3oclgtqy55fbzdi" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"cl": "cl0-global"}'
```

With timelock (premium):

```bash
curl -X POST "https://api.pinarkive.com/v3/files/pin/bafybeigdyrzt5sfp7udm7hu76uh7y26nf3efuylqabf3oclgtqy55fbzdi" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"cl": "cl0-global", "timelock": "2026-12-31T23:59:59Z"}'
```

## Python

```python
import os
import requests

cid = "bafybeigdyrzt5sfp7udm7hu76uh7y26nf3efuylqabf3oclgtqy55fbzdi"
url = f"https://api.pinarkive.com/v3/files/pin/{cid}"

    "Authorization": f"Bearer {os.environ['PINARKIVE_API_KEY']}",
    "Content-Type": "application/json",
```

## Example response

**Success (200/201)**

```json
{
  "cid": "bafybeigdyrzt5sfp7udm7hu76uh7y26nf3efuylqabf3oclgtqy55fbzdi",
  "clusterId": "cl0-global",
  "expiresAt": "2026-12-31T23:59:59.000Z"
}
```

| Field       | Type   | Description                          |
|------------|--------|--------------------------------------|
| `cid`      | string | The pinned CID                      |
| `clusterId`| string | Cluster where the content was pinned|
| `expiresAt`| string \| null | Expiration (timelock), if set |

**Error (4xx/5xx)**

```json
{
  "error": "Bad Request",
  "message": "Invalid CID"
}
```

Common codes: **401** (unauthorized), **403** (e.g. quota, timelock on free plan, or **missing scope**: `code: missing_scope`, `required`), **404** (CID not found), **429** (rate limit; `retryAfter`, header `Retry-After`).
